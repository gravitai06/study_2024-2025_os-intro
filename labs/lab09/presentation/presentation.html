##Вставил рандомный текст

Удаление строки:

Ctrl + Y — удалить текущую строку.

Выделение:

Ctrl + \ → стрелками выделите текст → Enter.

Копирование:

F5 → Enter.

Сохранение:

F2.

Отмена действия:

Ctrl + U.

Переход в конец файла + текст:

Ctrl + End

Переход в начало файла + текст:

Ctrl + Home

Сохранение и выход:

F2 → F10
